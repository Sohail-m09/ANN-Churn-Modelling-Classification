🏦 Customer Churn Prediction using Artificial Neural Networks (ANN)

This project is an end-to-end Customer Churn Prediction System developed using Artificial Neural Networks (ANN) with TensorFlow and Keras. The model predicts whether a bank customer is likely to leave the bank based on customer demographics, account details, and banking behavior. The project demonstrates the complete deep learning workflow, including data preprocessing, feature engineering, model training, evaluation, and deployment using Streamlit.

A user-friendly web application has been built with Streamlit, allowing users to input customer details and instantly receive a churn prediction along with the probability score. This project highlights practical implementation of deep learning for binary classification problems and serves as a portfolio-ready example for Machine Learning and AI roles.

🎯 Project Objective

The primary objective of this project is to leverage Deep Learning techniques to accurately identify customers who are likely to leave the bank. Early churn prediction enables organizations to improve customer retention strategies, reduce revenue loss, and enhance customer satisfaction through proactive engagement.
