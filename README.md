# 🏦 Customer Churn Prediction using Artificial Neural Networks (ANN)

An end-to-end **Customer Churn Prediction System** developed using **Artificial Neural Networks (ANN)** with **TensorFlow** and **Keras**. The model predicts whether a bank customer is likely to leave the bank based on customer demographics, account details, and banking behavior.

The project demonstrates the complete Deep Learning workflow, including data preprocessing, feature engineering, model training, evaluation, and deployment using **Streamlit**.

A user-friendly web application has been built with Streamlit that allows users to enter customer information and instantly receive a churn prediction along with the probability score.

# ✨ Features

- Predicts customer churn using an Artificial Neural Network (ANN)
- Interactive Streamlit web application
- Real-time churn prediction
- Displays prediction probability
- Data preprocessing using Label Encoding, One-Hot Encoding, and StandardScaler
- Deep Learning model built with TensorFlow and Keras
- Trained model stored in HDF5 (.h5) format
- End-to-end deployment on Streamlit Community Cloud
